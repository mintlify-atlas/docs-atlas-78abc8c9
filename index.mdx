  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Expensia in minutes
  </p>

  <Steps>
    <Step title="Register a new user account">
      Create your first user account using the registration endpoint.

      ```bash cURL
      curl -X POST https://api.expensia.com/register \
        -H "Content-Type: application/json" \
        -d '{
          "name": "John Doe",
          "email": "john@example.com",
          "password": "securepassword123"
        }'
      ```

      <Accordion title="Response">
        ```json
        {
          "user_id": 1,
          "message": "User created successfully"
        }
        ```
      </Accordion>
    </Step>

    <Step title="Authenticate and get access tokens">
      Login with your credentials to receive JWT access and refresh tokens.

      ```bash cURL
      curl -X POST https://api.expensia.com/login \
        -H "Content-Type: application/json" \
        -d '{
          "email": "john@example.com",
          "password": "securepassword123"
        }'
      ```

      <Accordion title="Response">
        ```json
        {
          "user_id": 1,
          "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
          "refresh_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
        }
        ```
      </Accordion>
    </Step>

    <Step title="Create your first transaction">
      Start tracking your finances by creating a transaction.

      ```bash cURL
      curl -X POST https://api.expensia.com/transactions \
        -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
        -H "Content-Type: application/json" \
        -d '{
          "transaction_type": "expense",
          "amount": 50.00,
          "note": "Grocery shopping",
          "category_id": 1,
          "transaction_date": "2026-03-04T10:00:00"
        }'
      ```

      <Accordion title="Response">
        ```json
        {
          "transaction_id": 1,
          "current_balance": -50.00
        }
        ```
      </Accordion>
    </Step>

    <Step title="Retrieve your transactions">
      View all your financial transactions and monitor your balance.

      ```bash cURL
      curl -X GET https://api.expensia.com/transactions \
        -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
      ```

      Learn more in the [Transactions Guide](/guides/transactions).
    </Step>
  </Steps>
</div>
